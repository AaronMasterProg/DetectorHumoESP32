# Sensor de temperatura con buzzer (Arduino)

Objetivo: Esta simulación tiene el fin de orientar al estudiante respecto al alambrado físico que tendrá que realizar para el detector de humo en ESP32.

Este proyecto simula un sensor de temperatura en Arduino usando Tinkercad.
El buzzer varía su sonido según el valor leído por una entrada analógica.
